## Publication

Navid Salehnamadi, Abdulaziz Alshayban, Jun-Wei Lin, Iftekhar Ahmed, Stacy Branham, and Sam Malek. 2021.LATTE: Use-Case andAssistive-Service Driven Automated Accessibility Testing Framework for Android.
